# MCP とは何か？
- **Model Context Protocol** の略  
- DCC（Blender, AE, Unreal など）間で  
  - メタデータ共有  
  - コマンド発行  
  - 状態同期  
- Python SDK ＋ WebSocket ベース

---

# 解決したい課題
**「分からないことを簡単に調べる」**  
- 手元の Blender 操作を中断せずに  
- ドキュメント・社内ナレッジ・Web情報を即取得  
- フローの途切れを最小化
